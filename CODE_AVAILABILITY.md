# Code Availability

Analysis scripts used to prepare the data products, fit the fixed-effects
models, run robustness checks and summarize the core evidence are provided in
this repository.

Suggested manuscript wording:

> The analysis code used to generate the crop portfolio metrics, drought
> exposure diagnostics, fixed-effects models, robustness analyses, nutrition and
> value-weighted outcome analyses, and crop-response asynchrony diagnostics is
> available at [GitHub repository URL]. Large input and intermediate datasets are
> described in the Data Availability statement.
